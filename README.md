# 💥 捏爆“马虫” - 3D 手势交互找不同游戏

这是一个基于浏览器运行的 3D 物理 + AI 手势识别的创意小游戏。在一场不断掉落的方块雨中，找出唯一带有“马虫”字样的方块，并通过真实的**手部握拳动作**将其捏爆，享受极具张力的膨胀撑破与炫彩粒子爆炸特效！

## ✨ 核心亮点

- 🖐️ **AI 实时手势捕捉**：基于 MediaPipe Hands，隔空操作，张开手掌瞄准，握拳确认捏爆。
- 🧊 **硬核 3D 物理反馈**：基于 Cannon.js 实现真实的重力、碰撞、堆叠与受力弹跳。
- 🎆 **炫酷的视觉特效**：Three.js 渲染高光阴影，目标方块被击中后会产生“吸水膨胀 -> 圆润撑破 -> 炫彩霓虹粒子炸裂”的爽快表现。
- ⚡ **极简纯粹，开箱即用**：零构建工具，所有的逻辑、渲染与 AI 模型拉取全部集成在一个 `index.html` 文件中。

## 🎮 玩法说明

1. **允许摄像头权限**：进入游戏后，请在浏览器弹窗中允许摄像头访问。
2. **张开手掌（瞄准）**：对着摄像头移动手掌，控制屏幕上的绿色小球准星。当准星停留在方块上时，方块会发光提示。
3. **用力握拳（捏爆）**：
   - ❌ **选错（冯强）**：方块会变成红色并剧烈弹跳抖动提示错误。
   - ✅ **选对（马虫）**：方块会迅速膨胀成球，最终撑破炸裂出大量炫彩粒子，并自动进入下一关（难度递增）。

## 🛠️ 技术栈

- [Three.js](https://threejs.org/) - 3D 场景渲染与炫彩爆炸粒子特效
- [Cannon-es](https://pmndrs.github.io/cannon-es/) - 3D 刚体物理引擎
- [MediaPipe Hands](https://developers.google.com/mediapipe) - 实时手部关键点 AI 检测
-原生 HTML5 / Canvas 2D API 动态生成材质

## 🚀 运行与部署指南

由于浏览器安全策略限制，网页**调用摄像头必须在 HTTPS 协议或本地 localhost 环境下**运行。
